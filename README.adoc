= Vert.x Maven Project


== Prerequisites

* Apache Maven
* JDK 8+

== Getting started

== Running the project

Run `redeploy.sh` (or `redeploy.bat` on Windows) and the project recompiles while editing.

== Building the project

To build the project:

----
mvn clean package
----



It generates a _fat-jar_ in the `target` directory.
